context.watch<BlocA>()
    - From the widget that was built within the context BuildContext
    - Start searching for the unique instance of BlocA(), provided above in the widget tree.
    - After its found, watch or subscribe to its stream of emitted states
    - And whenever a new state is emitted by BlocA, Rebuild the widget from which the lookup was started.

context.select
    - Useful when only wants to rebuild widget on simple select condition.
    - Scenario - Update a widget only when the name of the user changes
    - Equivalent to using buildWhen in BlocBuilder
